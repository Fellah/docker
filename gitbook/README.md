# GitBook Image

Run container:

```
$ docker run -p 80:4000 -v /srv/gitbook fellah/gitbook
```

`4000` – GitBook default service port.

`35729` – Live reload server port.

`/srv/gitbook` – Default working directory for GitBook container.
